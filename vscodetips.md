## **Terraform Commands in VS Code**

Press `Ctrl + Shift + P` and type:

- `Terraform: Init` - Initialize Terraform
- `Terraform: Plan` - Run terraform plan
- `Terraform: Validate` - Validate configuration

## **AWS Toolkit Features**

1. **View AWS Resources**:

   - Open AWS Explorer (AWS icon in sidebar)
   - Browse S3, Lambda, CloudFormation, etc.

2. **CloudWatch Logs**:

   - Right-click on log group in AWS Explorer
   - Select "View Log Stream"

3. **IAM Credentials**:
   - `Ctrl + Shift + P` → "AWS: Select AWS Profile"
